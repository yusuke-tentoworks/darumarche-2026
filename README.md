# Antigravity Web制作ルールブック

このプロジェクトは、AI（Antigravity）と連携して高品質なWebサイトを制作・運用するための標準ガイドライン集です。

## 📁 構成ファイル

各ガイドラインは、制作のフェーズに合わせて番号順に確認してください。

1. **[01_security_guidelines.md](01_security_guidelines.md)**: セキュリティの基本。開発着手前に必読。
2. **[02_seo_performance_guidelines.md](02_seo_performance_guidelines.md)**: 内部SEO・表示速度・SNS対策。設計時に確認。
3. **[03_coding_management_guidelines.md](03_coding_management_guidelines.md)**: フォルダ構成・命名規則・整理術。制作中のルール。
4. **[04_operation_legal_guidelines.md](04_operation_legal_guidelines.md)**: プライバシーポリシー・法務・ドメイン管理。公開準備に必要。
5. **[05_quality_assurance_guidelines.md](05_quality_assurance_guidelines.md)**: 最終チェック・品質検証。公開直前のチェックリスト。

---

## 🚀 運用ルール（使い方）

### 1. 新規プロジェクト開始時
- `PROJECT_MEMO_TEMPLATE.md` を、各プロジェクトの管理ディレクトリにコピーして使用してください。
- コピーしたファイルの名前を `PROJECT_MEMO.md` に変更し、プロジェクト情報を記入してください。

### 2. Antigravity（AI）とのやり取り
- このルールブックのディレクトリを常に Antigravity に読み込ませた状態で指示を出してください。
- 「ガイドラインの〇番に則って修正して」と指示することで、一貫した品質を保てます。

### 3. ルールの更新
- 制作を通じて得た新しい知見や、新しい技術標準があれば、随時これらのガイドラインをアップデートしてください。
